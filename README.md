# Prédiction du Churn Client

## Objectif
Prédire si un client va quitter l'entreprise à partir de ses données.

## Dataset
Dataset Telco Customer Churn (Kaggle)

## Méthodologie
- Analyse exploratoire
- Nettoyage des données
- Encodage des variables
- Modélisation (Logistic Regression)
- Évaluation du modèle

## Résultats
Le modèle atteint une accuracy satisfaisante sur les données de test.

## Outils
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
